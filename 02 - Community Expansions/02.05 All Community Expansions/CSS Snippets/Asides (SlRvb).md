##### Markdown Supported:
```md
Text
**Bolded Text**
~~Strikethrough Text~~

[[Links]]
`Inline code`

<i>Personal `<i></i>` adjustments</i>

Images: ![[Where on Sovera.svg|invertb]]

```

##### Not Supported Markdown:
- Lists
- Headings
- Code Blocks
- Blockquotes
